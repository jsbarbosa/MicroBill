microbill.constants Module¶

Variables¶

OLD_DIR

carpeta donde se guardan las cotizaciones realizadas

PDF_DIR

carpeta donde se guardan los PDFs asociados a cotizaciones realizadas

REGISTERS_DIR

carpeta donde se encuentran los archivos de registro de Microbill (Excel de los que se alimenta)

CLIENTES_FILE

archivo en el que se guarda el registro de clientes

REGISTRO_FILE

archivo en el que se guarda el resumen de la información de las cotizaciones

PRECIOS_FILE

archivo que contiene los precios de los servicios a usar

BASE_DIR

directorio base desde el cual se está ejecutando el intérprete de Python

EQUIPOS_KEYS

columnas que deben tener el archivo de precios

REGISTRO_KEYS

columnas que debe tener el archivo de registro

CLIENTES_KEYS

columnas que debe tener el archivo de clientes

DOCUMENTOS_FINALES

posibles formas de pago de una cotizacion

EQUIPOS

almacena el nombre de las hojas del archivo de precios

PRICES_DIVISION

categorias que dividen los precios de los servicios

REPORTE_INTERNO

nombre del archivo en donde se almacenan los reportes internos

DEFAULT_CONFIG

configuracion por defecto
