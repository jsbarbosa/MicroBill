# Install
## Virtual environment
On the working microbill directory
```
python -m venv .venv
```

Then, if Windows:
```
.venv\Scripts\activate
```

Else:
```
source .venv/bin/activate
```

## Requirements
```
pip install -r requirements.txt
```
